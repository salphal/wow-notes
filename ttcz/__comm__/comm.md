### 装备栏 - 使用

| 命令        | 描述       |
|-----------|----------|
| `/use 1`  | 头部       |
| `/use 2`  | 颈部( 项链 ) |
| `/use 3`  | 肩部       |
| `/use 4`  | 衬衣       |
| `/use 5`  | 胸甲       |
| `/use 6`  | **腰带**   |
| `/use 7`  | 腿部       |
| `/use 8`  | **鞋子**   |
| `/use 9`  | 护腕       |
| `/use 10` | **手套**   |
| `/use 11` | 戒指1      |
| `/use 12` | 戒指2      |
| `/use 13` | **饰品1**  |
| `/use 14` | **饰品2**  |
| `/use 15` | 披风       |
| `/use 16` | 主手武器     |
| `/use 17` | 副手/盾牌    |
| `/use 18` | 远程武器     |

---

| 命令             | 描述       | 正确语法示例              |
|----------------|----------|---------------------|
| `/cast`        | 释放技能     | `/cast 寒冰箭`         |
| `/cast`        | 释放指定等级技能 | `/cast 寒冰箭(等级 5)`   |
| `/stopcasting` | 取消当前施法   | `/stopcasting`      |
| `/cancelaura`  | 取消增益效果   | `/cancelaura 真言术：盾` |

---

| 命令                                                       | 描述                | 正确语法示例                                                          |
|----------------------------------------------------------|-------------------|-----------------------------------------------------------------|
| /cast [@focus,harm,nodead][@target,harm,nodead][harm]    | 焦点 > 目标 > 最近的敌方目标 | `/cast [@focus,harm,nodead][@target,harm,nodead][harm] 寒冰箭`     |
| /cast [@focus,help,nodead][@target,help,nodead][@player] | 焦点 > 目标 > 自己      | `/cast [@focus,help,nodead][@target,help,nodead][@player] 自由之手` |
